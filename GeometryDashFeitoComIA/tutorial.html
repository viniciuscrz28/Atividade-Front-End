Crie um Geometry Dash Minimalista em JavaScript

Neste tutorial você vai aprender como criar um jogo procedural inspirado em Geometry Dash usando apenas HTML, CSS e JavaScript puro. Sem frameworks, sem bibliotecas e sem dependências. Só o velho e poderoso navegador fazendo magia.

1. Criando o Canvas

Todo jogo web moderno começa com um canvas. Ele funciona como uma folha em branco onde desenhamos tudo: jogador, obstáculos, partículas e efeitos.

Código
<canvas id="game"></canvas>

const canvas = document.getElementById('game');
const ctx = canvas.getContext('2d');

canvas.width = window.innerWidth;
canvas.height = window.innerHeight;
Resultado
O segredo aqui é simples: o canvas é literalmente o monitor do seu jogo. Tudo será desenhado manualmente nele.

2. Criando o Player

O player é apenas um objeto JavaScript contendo posição, velocidade e tamanho. O truque está em atualizar esses valores constantemente.

Código
const player = {
  x: 180,
  y: 300,
  size: 42,
  velY: 0,
  grounded: false,
  rotation: 0
};
Preview

3. Física e Gravidade

Gravidade em jogos 2D é extremamente simples. Você só adiciona força vertical continuamente. Isso cria o efeito de queda.

Código
player.velY += gravity;
player.y += player.velY;

if(player.y >= groundY) {
  player.y = groundY;
  player.velY = 0;
  player.grounded = true;
}
Entendendo

• gravity = força constante
• velY = velocidade vertical
• y = posição do player

Jogos clássicos usam exatamente esse conceito desde os anos 80. O simples ainda domina.

4. Obstáculos Procedurais

O procedural deixa o jogo infinito. Em vez de criar fases manualmente, usamos números aleatórios.

Código
function spawnObstacle() {
  const h = Math.random() * 80 + 40;

  obstacles.push({
    x: canvas.width,
    y: groundY - h,
    w: 40,
    h
  });
}
Preview

5. Partículas Neon

Partículas são o que fazem o jogo parecer premium. Mesmo usando apenas quadrados pequenos, o cérebro interpreta como energia e movimento.

Código
particles.push({
  x: player.x,
  y: player.y,
  vx: Math.random() * 4 - 2,
  vy: Math.random() * -4,
  life: 30
});
Resultado

✦ ✦ ✦

Glow + partículas = sensação moderna.

6. O Game Loop

O coração do jogo é o loop principal. Ele atualiza a tela dezenas de vezes por segundo.

Código
function update() {
  requestAnimationFrame(update);

  ctx.clearRect(0, 0, canvas.width, canvas.height);

  drawPlayer();
  drawObstacles();
  updatePhysics();
}

update();
Explicação

requestAnimationFrame() sincroniza o jogo com os frames do navegador.

É isso que faz o jogo parecer fluido.
